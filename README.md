# My Test Site

A simple test website to learn how web development works!

## What's Included

This project contains:
- **index.html** - The main home page
- **clock.html** - Digital clock with multiple time zones
- **styles.css** - Styling for home page
- **clock-styles.css** - Styling for clock page
- **script.js** - JavaScript for home page
- **clock-script.js** - JavaScript for clock functionality

## Features

- 📱 Responsive design that works on desktop and mobile
- 🎨 Modern styling with gradients and transitions
- ✨ Interactive elements (button clicks, form submission)
- 🕐 Digital clock with 14+ time zones
- 🔗 Smooth scrolling navigation

## How to Use

1. Clone this repository to your computer
2. Open `index.html` in your web browser for the home page
3. Click on "Clock" to view the digital clock with multiple time zones

## Pages

### Home Page (index.html)
- Welcome section with interactive button
- About section explaining the site
- Contact form for user interaction

### Clock Page (clock.html)
- Displays current time in 4 default time zones
- Add new time zones from dropdown menu
- Remove individual time zones
- Reset to default view
- Remembers your selections using local storage
