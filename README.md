# My Test Site

A simple test website to learn how web development works!

## What's Included

This project contains:
- **index.html** - The main HTML structure with multiple sections
- **styles.css** - Styling and responsive design
- **script.js** - JavaScript for interactivity
- **README.md** - This file

## Features

- 📱 Responsive design that works on desktop and mobile
- 🎨 Modern styling with gradients and transitions
- ✨ Interactive elements (button clicks, form submission)
- 🔗 Smooth scrolling navigation

## How to Use

1. Clone this repository to your computer
2. Open `index.html` in your web browser
3. Explore the page and try clicking the button or filling out the contact form

## Learning Points

This site demonstrates:
- HTML structure with semantic elements
- CSS layout and styling techniques
- Basic JavaScript interactivity
- Responsive design with media queries
- Form handling

## Next Steps

Try modifying:
- Change the colors in `styles.css`
- Add more sections to `index.html`
- Create new JavaScript functions in `script.js`
- Add images or other media

Happy learning! 🚀
